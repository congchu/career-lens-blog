## 들어가며

이전 글에서 원티드 프론트엔드 공고 101개를 분석했을 때, **15개 공고가 AI 도구 역량을 요구**하고 있었습니다. 그중 가장 많이 언급된 도구는 Claude(5개), Codex(3개), Cursor(2개) 순이었죠.

> "Claude Code, Codex 등 LLM 도구를 개발 과정에서 적극적으로 활용해본 경험" — 소프트스퀘어드
> "AI 도구 마스터리: Cursor/Claude/Gemini/MCP 활용" — 메라키플레이스

그런데 막상 "AI 코딩 도구를 써보자!" 하면, 뭐부터 써야 할지 막막하죠.

Cursor? Copilot? Claude? ChatGPT? Codex? — 다 비슷해 보이는데 뭐가 다른 건지.

**신입/취준생 관점에서**, 각 도구가 뭘 하는 건지, 얼마인지, 뭘 먼저 써야 하는지 정리해봤습니다.

## 1. Claude — 채용공고가 가장 많이 찾는 도구

### 이게 뭔데?

Anthropic이 만든 AI입니다. 웹(claude.ai)에서 대화하면서 코드를 짤 수도 있고, **Claude Code**라는 터미널 기반 코딩 에이전트를 쓸 수도 있어요.

### 왜 공고에서 1위야?

Claude Code가 핵심입니다. 일반 챗봇처럼 "이거 만들어줘" 하는 수준이 아니라, **프로젝트 폴더를 통째로 이해하고 파일을 직접 만들고 수정**합니다. 

- 프로젝트 구조를 파악하고 여러 파일을 동시에 수정
- 터미널 명령어를 직접 실행 (빌드, 테스트, git 등)
- 기존 코드 스타일에 맞춰서 작성

공고에서 "Claude Code, Codex 등 LLM 도구를 적극 활용" 이라고 쓸 때, 이런 수준의 활용을 기대하는 겁니다.

### 가격

- **무료**: 일일 사용량 제한 있음 (가볍게 써보기 가능)
- **Pro $20/월**: Claude Code 포함, 충분한 사용량
- **Max $100/월**: 5배 사용량 (헤비 유저용)

### 신입에게 추천도: ★★★★★

**무조건 써보세요.** 무료로 시작할 수 있고, 공고에서 가장 많이 언급되는 도구입니다. Claude Code를 써보면 "AI로 개발한다"는 게 뭔지 바로 체감할 수 있어요.

## 2. Cursor — AI가 내장된 코드 에디터

### 이게 뭔데?

VS Code를 기반으로 만든 AI 코드 에디터입니다. VS Code랑 거의 똑같이 생겼는데, AI 기능이 에디터에 통합되어 있어요.

### 뭐가 좋은데?

- **Tab 자동완성**: 코드 치다가 Tab 누르면 AI가 다음 코드를 제안
- **Agent 모드**: "이 컴포넌트에 로딩 상태 추가해줘" 같은 명령으로 코드 수정
- **코드베이스 이해**: 내 프로젝트 전체를 맥락으로 잡고 답변
- **여러 AI 모델 선택**: Claude, GPT-4o, Gemini 중 골라서 사용 가능

핵심은 **에디터를 떠나지 않고 AI를 쓸 수 있다**는 점입니다. 브라우저에서 ChatGPT 열고, 코드 복사하고, 붙여넣고... 이런 번거로움이 없어요.

### 가격

- **Hobby (무료)**: Agent 요청 제한, 자동완성 제한
- **Pro $20/월**: 무제한 자동완성, 확장된 Agent
- **Pro+ $60/월**: 3배 사용량

### 신입에게 추천도: ★★★★★

VS Code 쓰고 있다면 **Cursor로 갈아타는 것만으로** AI 코딩이 시작됩니다. 설정이나 익스텐션도 그대로 옮겨지고, 단축키도 같아요. 진입 장벽이 거의 없습니다.

## 3. GitHub Copilot — 가장 먼저 등장한 AI 코딩 도구

### 이게 뭔데?

GitHub(Microsoft)이 만든 AI 코딩 어시스턴트입니다. VS Code, JetBrains 등 기존 IDE에 플러그인으로 설치해서 씁니다.

### 뭐가 좋은데?

- **코드 자동완성**: 코드 치다가 회색으로 뜨는 제안을 Tab으로 수락
- **Copilot Chat**: IDE 안에서 AI와 대화
- **Agent 모드**: Copilot도 최근 에이전트 기능 추가 (코드 수정, 터미널 실행)
- **GitHub 통합**: PR 리뷰, 이슈 분석 등 GitHub 워크플로우와 연결

가장 큰 장점은 **가격**입니다. 무료 플랜에서도 월 50회 프리미엄 요청을 쓸 수 있고, Pro가 $10/월로 다른 도구의 절반이에요.

### 가격

- **Free**: 월 50회 프리미엄 요청 + 자동완성
- **Pro $10/월**: 확장된 사용량 (학생/오픈소스 기여자 무료!)
- **Pro+ $39/월**: 에이전트 + 더 많은 모델

### 신입에게 추천도: ★★★★☆

**학생이라면 무료**로 Pro를 쓸 수 있어서 가성비 최고입니다. 다만 Cursor에 비해 AI 기능의 깊이가 조금 얕은 편이에요. "자동완성+가끔 채팅" 수준이면 Copilot, "AI와 깊이 협업"하려면 Cursor가 더 나아요.

## 4. ChatGPT — 가장 친숙한 AI

### 이게 뭔데?

OpenAI의 대화형 AI입니다. 모르는 사람이 없죠. 코딩 전용 도구는 아니지만, 코딩에 많이 쓰입니다.

### 코딩에 어떻게 쓰나?

- **코드 작성**: "React로 todo 앱 만들어줘" → 전체 코드 생성
- **디버깅**: 에러 메시지 복붙하면 원인과 해결책 제시
- **코드 리뷰**: "이 코드 문제점 찾아줘" → 개선점 제안
- **Canvas**: 코드를 실시간으로 편집하면서 AI와 협업하는 기능

### 한계

- **프로젝트 맥락을 모름**: 매번 코드를 복사해서 붙여넣어야 함
- **파일을 직접 수정 못 함**: 결과물을 사람이 옮겨야 함
- **IDE와 분리**: 브라우저 ↔ 에디터 왔다갔다 해야 함

Claude Code나 Cursor처럼 프로젝트를 통째로 이해하고 직접 수정하는 게 안 되기 때문에, 코딩 도구로서는 효율이 떨어집니다.

### 가격

- **무료**: GPT-4o 제한적 사용
- **Plus $20/월**: 더 많은 사용량 + 고급 모델
- **Pro $200/월**: 무제한 (대부분 필요 없음)

### 신입에게 추천도: ★★★☆☆

이미 쓰고 있을 거예요. 디버깅이나 개념 학습에는 좋지만, **"AI 코딩 도구 경험"으로 이력서에 쓰기에는 약합니다.** 공고에서 원하는 건 ChatGPT로 질문하는 수준이 아니라, Claude Code/Cursor로 실제 코드를 만드는 경험이니까요.

## 5. 그 외 — 알아두면 좋은 도구들

### Codex (OpenAI)

OpenAI의 코딩 에이전트입니다. Claude Code와 비슷하게 코드베이스를 이해하고 작업합니다. 공고에서 "Claude Code, Codex"로 함께 언급되는 경우가 많아요. ChatGPT Pro($200/월)에 포함되어 있어 접근성은 좀 떨어지지만, GitHub 연동이 강점입니다.

### Windsurf (구 Codeium)

Cursor의 경쟁자로 떠오르는 AI 코드 에디터입니다. Cascade라는 에이전트 기능이 있고, 무료 플랜이 꽤 넉넉합니다. 아직 채용공고에 등장하진 않지만 알아두면 좋아요.

### v0 (Vercel)

프롬프트로 UI 컴포넌트를 만들어주는 도구입니다. "다크 테마 대시보드 만들어줘" → Next.js + Tailwind 코드 생성. 프론트엔드 프로토타이핑에 특히 유용해요.

### Bolt / Lovable

프롬프트만으로 전체 웹앱을 만들어주는 도구들입니다. 프로토타입이나 MVP를 빠르게 만들 때 쓸 수 있지만, 실무 코드 품질과는 거리가 있어요.

**1순위: Cursor 설치**
- VS Code 대신 Cursor를 기본 에디터로 쓰기
- Tab 자동완성만 써도 체감이 확 다름
- 무료 플랜으로 충분히 체험 가능

**2순위: Claude 가입**
- claude.ai에서 코딩 질문, 코드 리뷰 시작
- 복잡한 로직이나 아키텍처 상담에 탁월
- Claude Code 써보기 (터미널에서 바로 코딩)

### 1개월 후 (필요하면 유료 전환)

**상황별 추천:**
- 학생이다 → **Copilot Pro (무료)** 쓰세요
- 사이드 프로젝트 많이 한다 → **Cursor Pro ($20)** 가성비 최고
- AI로 깊이 있게 개발하고 싶다 → **Claude Pro ($20)** Claude Code 활용

### 이력서에 쓸 때

이렇게 쓰면 됩니다:

> "Cursor + Claude Code를 활용한 AI 기반 개발 워크플로우 구축"
> "AI 코딩 도구를 활용하여 컴포넌트 개발 시간 50% 단축"

"ChatGPT로 코드 물어봤어요"가 아니라, **"AI 도구로 실제 결과물을 만들었어요"**가 포인트입니다.

## 자주 묻는 질문

### Q. Cursor랑 Copilot 둘 다 써야 하나요?

**둘 중 하나만 쓰세요.** 둘 다 IDE에서 AI 자동완성을 해주는 도구라 기능이 겹칩니다. Cursor를 쓰면 Copilot은 필요 없어요. (Cursor가 내부적으로 Claude/GPT를 쓰기 때문에 상위 호환에 가깝습니다.)

### Q. Claude와 ChatGPT는 뭐가 다르나요?

**코딩에서는 Claude가 더 강합니다.** 특히 Claude Code는 터미널에서 프로젝트를 직접 수정하는 에이전트인 반면, ChatGPT는 브라우저 대화창에서 코드를 주고받는 수준이에요. 채용공고에서 Claude가 5배 더 많이 언급되는 데는 이유가 있습니다.

### Q. 무료로만 써도 되나요?

**네, 무료로 충분히 시작할 수 있습니다.** Cursor Hobby + Claude 무료 + Copilot Free를 조합하면 돈 한 푼 안 쓰고 모든 도구를 체험할 수 있어요. 유료는 사용량이 부족해질 때 전환하면 됩니다.

### Q. 혹시 AI 도구에 의존하면 실력이 안 느나요?

이건 진지하게 답하겠습니다. **"AI로 코드 짜기"와 "코드를 이해하기"는 다릅니다.** AI가 만든 코드를 그대로 쓰기만 하면 실력이 안 늘지만, AI가 만든 코드를 **읽고, 이해하고, 수정하는 과정**에서 오히려 학습 속도가 빨라집니다.

AI 도구는 "대신 해주는 것"이 아니라 "더 빠르게 배우고 만드는 것"으로 쓰세요.

## 마무리

정리하면 이렇습니다:

| 도구 | 한 줄 요약 | 써야 하는 이유 |
|------|-----------|---------------|
| **Claude** | 코딩 에이전트의 대세 | 공고 언급 1위, Claude Code가 핵심 |
| **Cursor** | AI가 내장된 에디터 | VS Code 대체, 가장 쉬운 시작점 |
| **Copilot** | 가성비 자동완성 | 학생 무료, 가격 최저 |
| **ChatGPT** | 만능 AI 비서 | 디버깅/학습용, 코딩 전용으로는 부족 |

**오늘 할 일: Cursor 설치 → Claude 가입 → 사이드 프로젝트에 적용**

이것만 하면 "AI 도구 쓸 줄 아세요?" 라는 질문에 "네" 라고 답할 수 있습니다.
