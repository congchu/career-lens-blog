## 들어가며

마음에 드는 공고를 찾았습니다. 기술 스택도 맞고, 연봉도 괜찮아 보이고.

그래서 잡플래닛을 검색했는데... **후기 0건.** 블라인드에도 없고. 기술 블로그도 없고.

> "이 회사 괜찮은 건가? 후기가 없으면 위험한 거 아닌가?"

**아닙니다.** 후기가 없는 이유는 여러 가지예요. 위험 신호일 수도 있고, 단순히 작은 회사라서일 수도 있습니다.

---

## 후기가 없는 이유 3가지

### 이유 1: 회사가 작아서 (가장 흔함)

101개 공고의 회사 규모:

| 규모 | 비율 |
|------|------|
| **50명 이하** | 37% |
| 51~300명 | 24% |
| 300명 이상 | 나머지 |

**37%가 50명 이하입니다.** 10~20명 규모의 회사에서는:
- 후기를 써봤자 **누가 썼는지 바로 특정**됨
- 잡플래닛/블라인드에 **아예 등록이 안 되어 있기도** 함
- 직원 수가 적으니 후기 자체가 생길 수 없음

**이건 위험 신호가 아닙니다.** 그냥 작은 회사예요.

### 이유 2: 신생 기업이라서

설립 3년 이내인 회사는 아직 후기가 쌓이지 않았을 수 있어요.

| 설립 연차 | 공고 수 |
|-----------|---------|
| 설립 4~9년 | 42개 |
| 설립 10년 이상 | 28개 |
| (설립 연차 태그 없음) | 31개 |

태그 없는 31개 중 일부는 설립 1~3년 신생 기업.

### 이유 3: 정보를 통제하는 회사 (주의)

드물지만, **의도적으로 정보를 숨기는 회사**도 있습니다. 이건 경고 신호예요.

- 잡플래닛 부정 후기에 법적 대응
- 직원에게 후기 작성 금지 압박
- 회사명 검색해도 **뉴스, 블로그, SNS에 아무것도 안 나옴**

---

## 정보 없는 회사를 조사하는 7가지 방법

### 방법 1: 원티드 company_tags 확인

이전 편(F5)에서 다룬 것처럼, 원티드 태그만으로도 꽤 많은 정보를 얻을 수 있어요.

- "퇴사율 5% 이하" → 안정적
- "연봉 업계평균 이상" → 보상 괜찮음
- "인원급성장" → 사업 성장 중
- 태그가 거의 없음 → 추가 조사 필요

### 방법 2: 사업자등록 정보 확인

**국세청 홈택스**나 **공정거래위원회** 사이트에서 회사 정보를 무료로 확인할 수 있어요.

확인할 것:
- 실제 존재하는 법인인지
- 설립일
- 대표자
- 업종

### 방법 3: 투자 이력 확인

**더브이씨(THE VC)**, **크런치베이스**에서 투자 유치 이력을 확인하세요.

- 시리즈 A 이상 투자 유치 → 어느 정도 검증됨
- 유명 VC(소프트뱅크벤처스, 알토스벤처스 등)가 투자 → 긍정 신호
- 투자 이력 없음 → 자체 자금 or 아직 초기

### 방법 4: LinkedIn 검색

회사명으로 LinkedIn을 검색하면:
- 현직자/전직자 프로필
- 팀 규모 추정
- 개발자 경력 수준 (시니어가 있는지)
- 재직 기간 패턴 (1년 미만이 많으면 주의)

### 방법 5: GitHub/기술 블로그

회사 이름 + "tech blog" 또는 "github"로 검색.

- GitHub Organization이 있고 활발 → 기술 문화 긍정적
- 기술 블로그가 있고 최근 글이 있음 → 기술 투자 하는 회사
- 둘 다 없음 → 기술 공유에 관심 없거나 여력이 없음

### 방법 6: 뉴스 검색

네이버 뉴스에서 회사명 검색.

- 투자 유치 뉴스 → 긍정
- 수상/파트너십 뉴스 → 긍정
- 소송/논란 뉴스 → 경고
- 뉴스 자체가 없음 → 작은 회사 (중립)

### 방법 7: 면접에서 직접 확인

면접은 **회사가 나를 평가하는 자리이자, 내가 회사를 평가하는 자리**입니다.

물어볼 질문:
> "프론트엔드 팀은 몇 명인가요?"
> "코드 리뷰 문화가 있나요?"
> "온보딩은 어떻게 진행되나요?"
> "이 포지션이 열린 이유가 뭔가요?" (증원 vs 충원)
> "최근 퇴사한 개발자가 있다면 퇴사 이유가 뭐였나요?"

마지막 질문이 부담스러우면 이렇게:
> "개발팀 평균 재직 기간이 어떻게 되나요?"

## "후기 없는 회사" 판단 매트릭스

| 조건 | 신호 | 판단 |
|------|------|------|
| 50명 이하 + 설립 4년 이상 + 퇴사율5% | 작지만 안정적 | **지원 OK** |
| 50명 이하 + 설립 1~3년 + 투자 유치 | 초기지만 성장 중 | **지원 OK (주의하며)** |
| 태그 거의 없음 + 뉴스 없음 + LinkedIn 없음 | 정보 부족 | **추가 조사 필요** |
| 부정 후기 삭제 의혹 + 채용 과다 | 위험 신호 | **패스 추천** |

## 정보가 없을 때의 마인드셋

### "정보 없음 ≠ 나쁜 회사"

많은 좋은 스타트업이 후기 0건입니다. 20명 회사에서 잡플래닛 후기를 쓰면 바로 특정되니까요.

### "면접 = 최고의 정보 수집"

서류 넣고 면접에 가보는 것 자체가 정보 수집입니다. 면접에서 팀 분위기, 기술 수준, 문화를 직접 확인할 수 있어요.

**면접 = 합격해야만 의미 있다?** 아닙니다. 떨어져도 **"이 회사는 이런 곳이구나"**를 알게 되니까 면접 경험 자체가 자산입니다.

### "아는 만큼 보인다"

이 시리즈를 읽었으면 공고에서 읽을 수 있는 정보가 훨씬 많아졌을 거예요:
- 기술 스택의 진위 (F1)
- 채용 프로세스 유형 (F2)
- 레드플래그 (B4)
- company_tags로 건강 상태 확인 (F5)

**후기가 없어도, 공고 자체에 정보가 있습니다.**

## 정리

| 오해 | 현실 |
|------|------|
| 후기 없음 = 위험 | 작은 회사라서 없는 경우가 대부분 |
| 후기로만 판단해야 함 | 공고 태그, 투자 이력, LinkedIn으로 교차 검증 |
| 정보 없으면 패스 | 면접에서 직접 확인하는 게 가장 정확 |
| 큰 회사만 안전 | 50명 이하 + 퇴사율5%이하 조합이 오히려 안정적 |

**정보가 없다고 기회를 놓치지 마세요.** 조사할 수 있는 만큼 조사하고, 면접에서 직접 확인하고, 최종 판단은 그때 하면 됩니다.
