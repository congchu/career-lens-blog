## 들어가며

면접 끝나고 처우 협의를 했습니다.

> "연봉은 3,500만 원이고, 스톡옵션 000주를 드립니다."

3,500은 좀 낮은데... 스톡옵션이 있으니까 괜찮은 건가?

**"스톡옵션 = 대박"이라는 환상을 갖고 계시면, 이 글을 꼭 읽어주세요.**

- 공개를 꺼리거나

그리고 **"연봉 협의"라고만 써있는 공고 = 대부분 낮은 편**입니다.

## 스톡옵션이란 무엇인가 (30초 설명)

> "미래에 회사 주식을 정해진 가격에 살 수 있는 권리"

예시:
- 행사가격: 주당 1,000원에 1,000주 = 100만 원
- 3년 후 회사 가치가 올라서 주당 10,000원이 되면
- 100만 원으로 산 주식이 1,000만 원 → **900만 원 이익**

**하지만 회사 가치가 안 오르면 → 0원**

---

## 스톡옵션의 현실

### 현실 1: 대부분의 스타트업 스톡옵션은 가치가 없습니다

통계적으로 **스타트업의 90%가 실패**합니다. 스톡옵션은 회사가 상장하거나 인수되어야 돈이 되는데, 그 확률이 극히 낮아요.

| 시나리오 | 확률 | 스톡옵션 가치 |
|---------|------|-------------|
| 회사 폐업 | ~70% | **0원** |
| 회사 생존, 성장 못 함 | ~20% | **0원 (못 팔음)** |
| 인수/합병(M&A) | ~8% | **일부 가치** |
| IPO (상장) | ~2% | **높은 가치 가능** |

### 현실 2: 행사 조건이 까다롭습니다

> "4년 베스팅, 1년 클리프"

이게 뜻하는 것:
- **1년 클리프**: 1년 이내 퇴사하면 스톡옵션 0
- **4년 베스팅**: 4년에 걸쳐 조금씩 행사 가능
- 1년 근무 → 25%, 2년 → 50%, 3년 → 75%, 4년 → 100%

**즉, 1년 안에 나가면 스톡옵션은 없는 것과 같습니다.**

### 현실 3: 행사해도 현금화가 어렵습니다

비상장 회사의 주식은 **팔 곳이 없어요.** 상장 전까지는 그냥 종이쪼가리입니다.

- 상장 = 주식 거래소에서 거래 가능 → 현금화 가능
- 비상장 = 팔 수 없음 (일부 세컨더리 마켓 제외)

### 현실 4: 세금이 있습니다

스톡옵션을 행사할 때 **소득세가 발생**합니다. 행사가격과 시가의 차이에 대해 세금을 내야 해요. 현금이 없는데 세금은 내야 하는 역설적 상황이 생길 수 있습니다.

---

## 그러면 스톡옵션은 의미 없나요?

**아닙니다.** 아래 조건이면 가치가 있을 수 있어요:

### 스톡옵션이 의미 있는 경우

| 조건 | 확인 방법 |
|------|----------|
| 시리즈 B 이상 투자 유치 | 뉴스, 크런치베이스 |
| 매출이 실제로 있음 | 공시자료, 뉴스 |
| IPO/M&A 계획이 구체적 | 면접에서 물어보기 |
| 유명 VC가 투자 | 투자자 명단 확인 |
| 업계에서 1~3위 | 시장 점유율 |

### 스톡옵션을 주의해야 하는 경우

| 경고 신호 | 의미 |
|-----------|------|
| "스톡옵션 있어요" (금액/주수 미공개) | 가치 불명 |
| 시드/시리즈A 단계 | 상장까지 5~10년 |
| 연봉이 시장가보다 20% 이상 낮으면서 스톡옵션으로 보전 | 현재 가치 < 미래 불확실성 |
| 베스팅 조건을 안 알려줌 | 확인 필수 |

---

## 신입 기준: 연봉 vs 스톡옵션 판단법

### 원칙 1: 연봉이 먼저입니다

스톡옵션은 **보너스**예요. 기본 연봉이 시장가 이하인데 스톡옵션으로 보전하려는 건 **"지금은 적게 주고 나중에 줄게"**라는 뜻입니다.

> 연봉 4,000 + 스톡옵션 > 연봉 3,500 + 스톡옵션

같은 스톡옵션이면 **연봉이 높은 쪽**이 항상 유리합니다.

### 원칙 2: 스톡옵션은 0원이라고 가정하세요

스톡옵션의 "기대값"으로 의사결정하지 마세요. **0원이어도 괜찮은 연봉인지**가 기준입니다.

> "연봉 3,500이어도 이 회사에서 배울 수 있는 게 많으니까 OK"
> → 이러면 스톡옵션은 보너스

> "연봉 3,500은 부족한데, 스톡옵션이 있으니까..."
> → 위험한 판단

### 원칙 3: 처우 협의에서 꼭 물어볼 것

| 질문 | 왜 물어야 하는지 |
|------|----------------|
| "스톡옵션 몇 주인가요?" | 주수를 모르면 가치 판단 불가 |
| "행사가격은 얼마인가요?" | 행사가 = 내가 사야 하는 가격 |
| "베스팅 기간과 클리프는?" | 언제부터 행사 가능한지 |
| "최근 밸류에이션은?" | 회사 가치 기준으로 현재 가치 추정 |
| "IPO 또는 M&A 계획은?" | 현금화 시점 예측 |

이걸 물어보는 게 **전혀 무례하지 않습니다.** 오히려 "이 사람 똑똑하네" 하고 생각할 거예요.

## 처우 협의 실전 팁

### "연봉 협의" 공고의 현실적 범위

프론트엔드 신입/주니어 기준 (2026년):

| 회사 유형 | 예상 범위 |
|-----------|----------|
| 대기업 | 4,500~5,500만 |
| 중견기업 | 3,800~4,500만 |
| 스타트업 (투자 유치) | 3,500~4,500만 |
| 소규모 스타트업 | 3,000~3,800만 |

**"연봉 업계평균 이상" 태그가 있는 31개 공고**가 상대적으로 안전한 선택이에요.

### 연봉 말고 확인할 보상

| 항목 | 확인 |
|------|------|
| 4대보험 | 기본 (없으면 불법) |
| 퇴직금 | 1년 이상 근무 시 |
| 연차 | 법정 15일 + 추가 여부 |
| 성과급/인센티브 | 고정급 외 변동급 |
| 교육비 | 컨퍼런스, 강의, 도서 |
| 장비 | 맥북, 모니터 지급 여부 |

## 정리

| 오해 | 현실 |
|------|------|
| 스톡옵션 = 대박 | 90%는 0원 |
| 낮은 연봉 + 스톡옵션 = OK | 연봉이 먼저 |
| 스톡옵션 물어보면 실례 | 당연히 물어봐야 함 |
| "연봉 협의" = 잘 주겠다 | 대부분 낮은 편 |

**스톡옵션은 "있으면 좋은 것"이지, "있어야 하는 것"이 아닙니다.** 기본 연봉으로 생활이 가능한지가 첫 번째 기준이고, 스톡옵션은 그 위에 올리는 보너스예요.

신입이라면: **연봉 + 성장 환경 + 팀 구성**을 먼저 보세요. 스톡옵션은 3년 뒤의 이야기이고, 3년 안에 배우는 것이 커리어에 훨씬 큰 영향을 미칩니다.
