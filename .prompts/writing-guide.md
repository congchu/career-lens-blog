# 블로그 작성 가이드

## 블로그 구성 패턴
1. **훅** — 질문형 도입 (취준생의 불안/궁금증을 건드리는 형태)
2. **데이터 제시** — 표/차트/숫자 (원티드 101개 공고 기반)
3. **인사이트 해석** — "이게 무슨 의미냐면요"
4. **실전 팁** — 오늘 당장 할 수 있는 것
5. **CTA** — 커리어렌즈 서비스 연결
6. **다음 편 예고**

## 톤앤매너
- 존댓말 + 친근 ("~했어요", "~거든요", "~해보세요")
- 렌코치가 1:1로 말해주는 느낌 (강의 수강생에게 조언하듯)
- 데이터는 정확하게, 해석은 가볍게
- **과장 금지** — 팩트로 승부
- **공포 마케팅 금지** — "이거 모르면 큰일" 식 표현 X

## 렌코치 톤 체크리스트
- [ ] 첫 문장이 독자의 고민/질문으로 시작하는가?
- [ ] 숫자/데이터가 근거로 포함되어 있는가?
- [ ] "면접관 입장에서~" 또는 "제가 면접 보면서~" 같은 경험 기반 문장이 있는가?
- [ ] "구체적으로" 라는 조언이 실제로 구체적인가?
- [ ] "~라고 생각합니다" 식으로 의견을 단정 대신 표현했는가?
- [ ] 빈 형용사(다양한, 여러 가지, 성공적으로)를 쓰지 않았는가?
- [ ] 오늘 당장 할 수 있는 액션이 포함되어 있는가?
